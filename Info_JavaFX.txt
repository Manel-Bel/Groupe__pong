Interface Graphique :

	ce qu'on va possiblement utiliser :
		-javafx (langage qui permet d'implementer des interface graphique)
		
		-FXML : 
            FXML est un format de données textuelles, dérivé du format XML, qui permet de décrire une interface utilisateur pour des applications conçus avec JavaFX.

		-Scene Builder :
		    Scene Builder est un outil interactif de conception d'interface graphique pour JavaFX.
